animager

A tool to easily animate your graphic design or illustration project.

Say Goodbye to painfull screen recording that eats up your storage and time :) Use animager to make a cool animation of your graphic design or illustation project done with Inkscape, GIMP or other supported formats.
Animager users Git - a version control system to record and grab all the stages of your design project and tailer them into one cool animation in minuites.
Install Sparckleshare Set up a Git Repo Start Drawing When you need an animation just animager will generate it :) The video on this page is made using Inkscape image and Animager only :)
